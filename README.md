# Global-Sales-Dashboard

##  Project Overview

This project presents a Global Sales Performance Dashboard created using Power BI. 
It analyzes sales transactions across different countries, regions, and markets to provide meaningful insights for stakeholders.

---

##  Objectives

- Clean and prepare raw sales data
- Segment data using Country, Region, and Market
- Analyze Shipping Mode percentage distribution
- Visualize Sales across City, State, Region, and Market
- Build an interactive dashboard with tables and filters

---

##  Data Cleaning

The dataset was cleaned before visualization:

- Removed duplicate values
- Handled missing/null data
- Corrected data types (Date, Numbers)
- Standardized column names
